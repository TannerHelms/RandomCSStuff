Create Element

Step 1: This creates html elements in the place above the hr.

Step 2: You first have to put the html tag in the tag place.

Step 3: Then you can choose what the id and class are in their respective spots.

Step 4: Then in the innerHTML spot you can choose what will go inside of your tag. An example is if you had a h1 tag then it's going to be the words that are displayed.

Step 5: Then you click Create Element.

Remove Last Element

Step 1: Follow the Create Element Steps above.

Step 2: Then click the remove last element button to remove the last element that was created.

Reset

When the reset button is clicked it completely resets all of the inputs to their original states.
